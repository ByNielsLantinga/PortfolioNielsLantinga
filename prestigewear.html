Prestigewear

Het project:

Iemand in mijn familie wil een kledingbedrijf starten. Daarom heb ik een website ontworpen. De website is nog niet online, omdat deze nog niet af is.

Bij het ontwerpen van deze website heb ik Figma gebruikt en vervolgens de site ontwikkeld met Visual Studio. Ik heb gekozen voor een witte achtergrond om de site er luxueus uit te laten zien, omdat dit beter bij het merk past. Ik heb ook een aantal gebruikerstests uitgevoerd om te beoordelen hoe de site werkt.

Programma's gebruikt in het project:

Photoshop, Figma, Visual Studio Code(html, css)

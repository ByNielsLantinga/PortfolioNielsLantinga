Livewall

Het project:

LiveWall wil een sprookjes generator maken. Hier heb ik met mijn projectgroep aan geholpen. We hadden allemaal een design gemaakt. Hier is er een van gekozen. Ook heb ik in de project voor het eerst met een API gewerkt. Dit vond ik heel lastig, maar heb het wel werkend gekregen.

Er zitten een aantal verschillende functies in het project. Het sprookje genereren, achtergrondmuziek, afbeeldingen genereren en tekst to speech. De laatste functie was mijn verantwoordelijkheid. Ik heb hiervoor de API van Eleven Labs gebruikt.

We hadden te horen gekregen dat het design moest passen bij de site van LiveWall. Het design wat ik heb gemaakt zie je in de afbeeldingen. Mijn design was nog niet helemaal af, maar omdat er vroegtijdig een ander design gekozen was heb ik die van mij niet afgemaakt.

Programma's gebruikt in het project:

Photoshop, figma, Visual Studio Code(html, css)

Livewall

Het project:

LiveWall wil een sprookjes generator maken. Hier heb ik met mijn projectgroep aan meegeholpen. We hadden allemaal een ontwerp gemaakt. Hier is er een van gekozen. Ook heb ik in dit project voor het eerst met een API gewerkt. Dit vond ik heel lastig, maar heb het wel werkend gekregen.

Er zitten een aantal verschillende functies in het project. Het sprookje genereren, achtergrondmuziek, afbeeldingen genereren en tekst to speech. De laatste functie was mijn verantwoordelijkheid. Ik heb hiervoor de API van Eleven Labs gebruikt.

We hadden te horen gekregen dat het ontwerp moest passen bij de site van LiveWall. Het ontwerp wat ik heb gemaakt zie je in de afbeeldingen. Mijn ontwerp was nog niet helemaal af, maar omdat er vroegtijdig een ander ontwerp gekozen was, heb ik die van mij niet afgemaakt. Het bedrijf was blij met ons project en kon hiermee inspiratie opdoen voor hun eigen team.

Programma's gebruikt in het project:

Photoshop, figma, Visual Studio Code(html, css)

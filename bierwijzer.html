Het project:

Dit was mijn laatste project die ik op school heb gedaan. Het volgende semester ga ik stage lopen.

Dit project heeft 4 weken geduurd. We mochten zelf kiezen wat voor project we gingen doen. Ik vond het lastig om in het begin te kiezen wat ik nou wilde gaan maken. Uiteindelijk ben ik gegaan voor een website. Hierbij heb ik het volledige proces van een onderzoeken, ontwerpen en ontwikkelen nog eens doorgelopen. Het eindresultaat is links te zien. Eerst het wireframe en daarna het High-Fi prototype.

Ik vond het een leuk project om te doen. Ook heb ik hier veel van geleerd.
